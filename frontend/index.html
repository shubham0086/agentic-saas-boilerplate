DAG Engine Scheduler

Linear pipelines block execution speed. The zero-dependency DAG scheduler runs agents in parallel using Kahn's Algorithm. Planner executes first, Copywriter and ImageGenerator run concurrently, and Verifier triggers only when all upstream nodes report success.

FastAPI SSE Streaming

Timing Attack Guard

Malicious actors can deduce signature values byte-by-byte using timing analysis. We implement secure constant-time signature comparisons via hmac.compare_digest for both Stripe and Razorpay webhook validation.

Circuit Breaker Failover

API timeouts can lock your pipeline. The stateful circuit_breaker.py wrapper halts outbound calls to model providers after 3 consecutive errors, temporarily blocking requests and falling back to local mock streams during cooldown.

Kahn's Algorithm implementation dynamically calculates topological execution order, identifying parallel nodes.

core/workflow.py 
def topological_order(self) -> List[str]:
    """Kahn's algorithm — returns nodes in dependency order."""
    indeg: Dict[str, int] = defaultdict(int)
    adj:   Dict[str, List[str]] = defaultdict(list)

    for u, v in self.edges:
        adj[u].append(v)
        indeg[v] += 1
        indeg.setdefault(u, 0)

    q = deque([n for n in self.nodes if indeg.get(n, 0) == 0])
    order, seen = [], set()

    while q:
        u = q.popleft()
        if u in seen:
            continue
        seen.add(u)
        order.append(u)
        for w in adj[u]:
            indeg[w] -= 1
            if indeg[w] == 0:
                q.append(w)

Parses Stripe headers, structures expected signature payload, and verifies hashes in constant-time.

billing_router.py 
# Parse Stripe signature fields (e.g. t=timestamp,v1=sha256hash)
parts = {k: v for part in stripe_signature.split(',') for k, v in [part.split('=', 1)]}
timestamp = parts.get('t')
v1_hash = parts.get('v1')

# Recreate expected payload: timestamp + "." + raw_body
signed_payload = f"{timestamp}.".encode() + raw_body

# Calculate expected HMAC hash
expected_sig = hmac.new(
    STRIPE_SECRET.encode(),
    signed_payload,
    hashlib.sha256
).hexdigest()

# Constant-time comparison to protect against timing attacks
if not hmac.compare_digest(expected_sig, v1_hash):
    raise HTTPException(status_code=401, detail="Invalid signature hash")

State machine managing CLOSED, OPEN, and HALF_OPEN circuits to prevent backend network lockup.

circuit_breaker.py 
async def call(self, async_func: Callable[..., Any], *args, **kwargs) -> Any:
    self._check_recovery()

    if self.state == "OPEN":
        raise CircuitBreakerOpenException("Circuit breaker is OPEN. API calls disabled.")

    try:
        result = await async_func(*args, **kwargs)
        if self.state == "HALF_OPEN":
            self.state = "CLOSED"
            self.failure_count = 0
        return result
    except Exception as e:
        self.failure_count += 1
        self.last_failure_time = time.time()
        if self.state in ("CLOSED", "HALF_OPEN") and self.failure_count >= self.failure_threshold:
            self.state = "OPEN"
        raise e
